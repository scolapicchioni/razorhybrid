# Razor Pages

- **Index** - Razor page with list of all photos
- **Upload** - Razor page to create a new photo
- **Details** - Razor page to see an existing photo


## Lessons learned

- Dependency injection in Razor Pages
- Page Routing
- Page Actions
- Model Binding
